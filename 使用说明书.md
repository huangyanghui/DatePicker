# 参数
参数             说明                               类型       可选值               默认值
type            显示类型                            string     year/month/date      date
:pickeroptions   带快捷选项 ,type="date"，才生效     boolean    true/false           false
# 使用
1. 先把下载下来的文件夹放到vue-cli项目里
2. 在main.js中，import '文件夹路径'
3. 在.vue文件中直接使用 `` <su-datep></su-datep> ``
